# 신규 해역 추가 시 작업

### area 테이블에 레코드 추가
	- areaId, regionId, name, reqRank, exp, distance: 인게임에서 확인 가능한 정보 그대로 사용
	- time: 기준 잠수함의 순항 속도와 대상 해역의 탐사 시간을 곱하여 계산한 값
	  `e.g.) 순항 속도 175, A 남빛 물여울의 탐사 시간 652분 (10시간 52분)을 곱한 값인 114100을 입력
### navigation 테이블에 레코드 추가
	- areaFrom, areaTo, distance: 인게임에서 확인 가능한 정보 그대로 사용
	- time: 아래 각 케이스별로 기준 잠수함의 순항 속도와 대상 해역의 이동 시간을 곱하여 계산한 값 (공식은 상동)
		- 미탐사 해역 --> 해역 x 이동
		- 해역 x --> 해역 y 이동
		- 해역 y --> 해역 x 이동: 해당 건은 위 x --> y 케이스와 계산 값 자체는 동일하나, 빠른 계산을 위해 레코드를 추가하기 위함
### unlock 테이블에 레코드 추가
	- 다른 해역 해금 정보가 존재하는 경우에만 작업
	- areaFrom, areaTo: 인게임에서 확인 가능한 정보 그대로 사용
### rewards 테이블에 레코드 추가
	- areaId: 인게임에서 확인 가능한 정보 그대로 사용
	- tier: 잠수함 탐사 결과 보고서의 보상 획득 내용에 따라 0, 1, 2 입력
		- tier 0: "해저 화산이 활발하게 활동하고 있습니다...탐사 성능은 충분하지만 탐사에 큰 지장이 있습니다."
		- tier 1: "풍랑이 일고 있습니다...탐사 성능은 충분하지만, 탐사에 영향이 있습니다."
		- tier 2: "해류가 잔잔하여 절호의 조건입니다...{잠수함명}의 탐사 성능은 충분합니다."
	- itemId: [공식 가이드](https://guide.ff14.co.kr)에서 사용하는 아이템 고유식별자 입력
### items 테이블에 레코드 추가
	- 신규 보상이 추가된 경우에만 작업
	- itemId: [공식 가이드](https://guide.ff14.co.kr)에서 사용하는 아이템 고유식별자 입력
	- name: 인게임에서 확인 가능한 정보 그대로 사용
